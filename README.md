# PMS 2026 - Warehouse/Industrial Property Management System

A comprehensive property management system for managing warehouse and industrial properties with full accounting integration.

## Overview

This system provides comprehensive warehouse and industrial property management with QuickBooks-level accounting capabilities. Manage multiple warehouse facilities, track tenant leases, handle payments, and maintain complete double-entry bookkeeping.

## Features

- **Property Management**: Warehouse spaces, loading docks, and industrial facilities
- **Lease Management**: Track commercial tenants with flexible terms
- **Full Accounting**: Double-entry bookkeeping with trial balance verification
- **Invoice System**: Atomic invoice numbering with payment tracking
- **Payment Processing**: Multiple payment methods with automatic balance updates
- **Work Order Management**: Maintenance tracking for warehouse facilities
- **Financial Reports**: P&L statements, aging reports, and tenant balances
- **Document Library**: Store and manage lease agreements and property documents

## Tech Stack

- Next.js / TypeScript
- Prisma (Database ORM)
- Vercel (Deployment)
- Twilio (SMS API)

## Use Case

Designed for warehouse and industrial property management. Supports multiple facilities with tenant lease tracking, automated rent billing, payment processing, and comprehensive accounting. Ideal for logistics centers, distribution warehouses, and industrial parks.
