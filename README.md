# PMS 2026 - Residential Property Management System

A property management tool for managing 1-5 summer rental properties with SMS-based inquiry handling.

## Overview

This system helps small property owners manage rental inquiries by assigning dedicated Twilio phone numbers to each property. When potential renters text a property's number, inquiries are logged and managed through a simple dashboard.

## Features

- Dedicated phone number per property (via Twilio)
- SMS inquiry tracking and logging
- Property information management
- Inquiry response system
- Basic tenant contact management

## Tech Stack

- Next.js / TypeScript
- Prisma (Database ORM)
- Vercel (Deployment)
- Twilio (SMS API)

## Use Case

Designed for seasonal rental properties in New Jersey coastal areas. Supports 1-5 properties under single owner management with direct SMS communication to potential renters.
